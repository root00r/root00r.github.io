PIM C-RP Advertisement消息格式

C-RP Advertisement消息

当PIM-SM网络中使用动态RP时，配置了C-RP的路由器周期性的向BSR发送Advertisement消息，通告希望服务的组范围。

C-RP Advertisement消息只在PIM-SM中使用。

封装Advertisement消息的IP报文源地址为源端C-RP，目的地址为BSR。使用单播方式发送。

图1 Advertisement消息格式
字段 长度 说明
Version 4比特 PIM版本，值为2。
Type 4比特 消息类型，值为8。
Prefix-Cnt 8比特 组播地址前缀值。
Priority 8比特 C-RP优先级。
Holdtime 16比特 Advertisement消息的老化时间。
RP-Address 32比特 C-RP地址。
Group Address 32比特 组播组地址。
