图1 ICMP源端被关闭消息格式
字段 长度 含义
Type 1字节 消息类型，此处值为4。
Code 1字节 消息代码，此处值为0。
Checksum 2字节 检验和。
Internet Header + 64 bits of Original Data Datagram 可变 IP头和原始数据包的前64比特数据。该数据是主机用来匹配消息。对于更高层协议的用户端口号，原始数据包的前64比特的这些数据会被重组。
